# Non-Hermitian SSH Model

This project studies a non-Hermitian Su-Schrieffer-Heeger (SSH) tight-binding Hamiltonian.

## Physics

The Hamiltonian includes:
- asymmetric hopping
- non-Hermitian terms
- edge states
- complex energy spectra

## Features

- Complex eigenvalue spectrum
- Edge-state localization
- Open/Periodic boundary conditions

pip install -r requirements.txt

## Run

python run.py
